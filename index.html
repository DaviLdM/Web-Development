Meus Serviços

Hardware Description

Tenho experiência em descrição de hardware e backend, atuo no desenvolvimento de sistemas utilizando SystemVerilog, Verilog, SystemC e Python. Realizo modelagem de circuitos digitais, otimização de desempenho e automação de processos. Focado em qualidade e eficiência, ofereço soluções para projetos de hardware, desde a concepção até a verificação de hardware aplicada.

Full-Stack

Tenho experiência básica em desenvolvimento back-end com Java, onde construo APIs simples e funcionais. Também utilizo JavaScript para criar interações dinâmicas em interfaces e possuo conhecimentos em banco de dados, focando em segurança e boa organização de dados. Com noção em desenvolvimento web com HTML, CSS e JavaScript, criando sites responsivos com foco na funcionalidade e usabilidade do usuário.

FPGA

Tenho experiência em desenvolvimento com hardware, atuando com Arduino e ESP's para prototipagem rápida e FPGAs para sistemas personalizados de alta performance. Ultilizei o LiteX para criar SoCs para processadores, otimizando processamento e controle com experiencia no Quartus. Busco explorar soluções compactas e eficientes que integram hardware e software para aplicações avançadas e específicas.

Últimos Projetos

RISC-X

Processador RISC-V desenvolvido durante meu estágio com os colegas de equipe, suportando as extensões I, C e F (onde fiquei responsável), além das instruções padrão. O processador foi integrado em um SoC completo gerado pelo LiteX, fornecendo uma plataforma robusta para testes e otimizações. Esse projeto aprimora minha experiência em hardware e sistemas embarcados.

Banco de Dados

Participei de uma capacitação pela Accenture que passei com exito, desenvolvendo um projeto de banco de dados em Java com Spring. O objetivo era criar serviços REST eficientes e escaláveis, atendendo aos requisitos da empresa. Esse treinamento prático proporcionou experiência em desenvolvimento de back-end, reforçando habilidades em bancos de dados e APIs RESTful para o mercado corporativo.

Repositório VIRTUS

Repositório para documentar as soluções dos desafios Virtus, utilizando SystemVerilog para descrever cada atividade. Os desafios incluíram implementação de FFT em RTL, identificador de strings, operador Sobel e filtro passa-baixa. O projeto foi desenvolvido na plataforma EDAPlayground, com links para o código e um PDF explicando cada solução. Esses foram os desafios que me fizeram passar em primeiro lugar da seleção de alunos.

Carro Elétrico

Desenvolvi um projeto de carro eletrônico no laboratório de circuitos digitais, integrando dispositivos como acelerômetros, freios inteligentes, sensores de bateria e simulação de diâmetro de pneus. Todo o sistema foi projetado com circuitos digitais, usando SimulIDE e Quartus para simulação e implementação, visando maior eficiência e segurança no controle do veículo.

VIA-CEP

Desenvolvi um projeto de desafio de banco de dados, no qual, a partir de alguns dados e do CEP do usuário, é possível criar uma base de dados com diversas informações adicionais. Utilizei Java e Spring Boot para construir essa aplicação, visando ampliar minhas habilidades em backend e integrar APIs para enriquecer os dados armazenados.

Desenvolvimento Web

Desenvolvi este site para aprimorar minhas habilidades em desenvolvimento web, utilizando HTML, CSS e JavaScript, com foco em responsividade e características modernas de design. O projeto incluiu layouts adaptáveis e interações dinâmicas, proporcionando uma experiência de usuário agradável. Esse trabalho foi essencial para consolidar minha competência em frontend e desenvolvimento de sites atuais.
